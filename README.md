课程介绍：

机器学习算法和理论，是是以计算机学习人类行为为目的，而研究出的一整套方法论。这些方法论中，包括了如何从大量数据中挖掘规律和知识的方法论。推荐算法，是以给user推荐合适item为目的算法体系。而要实现这一目的，需要从大量数据中挖掘规律和知识，这就要用到机器学习这套方法论了。简单地说，机器学习是方法基础，推荐算法是器学习的各种机应用领域中的一种。
